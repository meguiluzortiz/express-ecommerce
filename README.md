# express-ecommerce

A simple ecommerce webapp made with express.

## How to run

Use npm tool and execute one of the next commands.

```(sh)
npm run start         #run the app
npm run start:debug   #run the app in debug mode
```

## Tech stack

- Node 12.18.0
- ExpressJS 4
- Json Web Token
- Mocha, Proxyquire, Sinon and Supertest
- Mongodb
- Vercel deploy
